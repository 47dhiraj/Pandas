
# str.contains(): The str.contains() method in Pandas is a string filtering method that allows you to select rows of a dataframe based on whether a string column contains a specified substring.


    ==> The str.contains() method takes a string or regular expression pattern as its argument and returns a boolean mask that indicates whether each element in a string column contains the specified substring.
        You can then use this boolean mask to filter the dataframe and select only the rows that contain the specified substring.


    # Example: # filter the dataframe based on a substring

                filtered_df = df[df['city'].str.contains('or')]         # yo line ko or chai filtering string or regular expression ho
