
# Here are some of the ways to write different types of files using Pandas:


1. to_csv(): to write data to a CSV file.

    # write the DataFrame to a CSV file
    eg: df.to_csv('mydata.csv', index=False)

    NOTE: The index=False parameter specifies that we don't want to write the row index to the file.



2. to_excel(): to write data to an excel file.

    # write the Dataframe to a Excel file
    eg: df.to_excel('mydata.xlsx', index=False)

    NOTE: The index=False parameter specifies that we don't want to write the row index to the file.



3. to_json(): to write data to JSON file.

    # write the DataFrame to a JSON file
    eg: df.to_json('mydata.json', orient='records')

    NOTE: The orient='records' parameter specifies that we want to write the data as a list of records.



4. to_sql(): to write dataframe to SQL database.

    NOTE: To write data to an SQL database, we first need to establish a connection to the database using a database engine.


    eg:
    # establish a connection to an SQL database
    engine = create_engine('sqlite:///mydata.db')

    # write the DataFrame to an SQL database
    df.to_sql('mytable', engine, index=False, if_exists='replace')
