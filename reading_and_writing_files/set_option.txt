
# pd.set_option() : `pd.set_option()` is a method in Pandas that allows you to set the value of a particular option in Pandas. 
                    These options control various behaviors in Pandas, such as the number of rows and columns to display, the precision of floating-point numbers, and the maximum width of the columns.


    # Note : By Default Jupyter notebook will display only first 5 rows and last 5 rows of dataframe.
            But, yo talako set_option() method ko help batw, aafno dataframe ma vayeko timle chayeko jati ota rows or columns herna milcha.



    SYNTAX: pd.set_option('display.max_rows', all_rows_number)
            eg:     df = pd.set_option('display.max_rows', all_rows)
            eg:     ef = pd.set_option('display.max.rows', 235)


    SYNTAX: pd.set_option('display.max_columns', all_columns_number)
            eg:     df = pd.set_option('display.max_columns', all_cols)
            eg:     df = pd.set_option('display.max.columns', 40)
