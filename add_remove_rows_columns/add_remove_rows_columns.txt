
# Here are some of the most commonly used Pandas methods to create new columns, drop columns, and split a single column into multiple columns in a Pandas DataFrame.


    1. Adding columns: 

        - `df['new_column'] = df['column_1'] + df['column_2']`: This creates a new column by adding two existing columns together.
        
        - `df['new_column'] = df.apply(lambda row: some_function(row['column_1'], row['column_2']), axis=1)`: This creates a new column by applying a custom function to each row of the DataFrame.
        
        - `df['new_column'] = some_value`: This creates a new column with a scalar value.



    2. Dropping columns:

        - `df.drop('column_name', axis=1, inplace=True)`: This drops a single column from the DataFrame.
        
        - `df.drop(['column_1', 'column_2'], axis=1, inplace=True)`: This drops multiple columns from the DataFrame.



    3. Splitting a single column into multiple columns:

        - `df[['new_col_1', 'new_col_2']] = df['column'].str.split('delimiter', expand=True)`: This splits a single column into two new columns using a delimiter (such as a comma).
        
        - `df['new_column'] = df['column'].apply(lambda x: x.split('delimiter')[index])`: This creates a new column by splitting a single column and selecting a specific element (as determined by `index`) from the resulting list.
